1. Easy First Action

Cold emails / network outreach

This is the lowest-friction action because it does not require travel, payment, or a big commitment.

First check:

  • Pick the first people or organisations to contact.
  • Send them even if the chance of reply is uncertain.

2. Sailing Check

RYA Competent Crew / sailing

This does not have to be a career decision. It can simply be a serious short test of sailing, crew life, responsibility, and whether this kind of environment gives energy.

First check:

  • Pick 2 or 3 dates or providers.
  • Check level fit: Competent Crew first, or a higher / more specific module if a provider says it makes sense.
  • Check cost, travel, and whether the dates fit around exams.

Useful links:

  • Tall Ships Youth Trust Start Yachting
  • Ocean Youth Trust South
  • UKSA Career Pathways

3. Serious Path To Keep Open

Ecole 42 family

This is useful because it can be tested in stages:

  • first, check fit through an intro / online step / nearby test, possibly Codam / 42 Amsterdam;
  • then, if it still looks interesting, consider a Piscine;
  • only after that, think about a full campus commitment.

First check:

  • Compare the 42 campuses that are actually realistic.
  • Check dates for intro meetings, Discovery / Piscine formats, and application steps.
  • Decide whether the first test should be nearby or in a city that is more exciting.

Useful links:

  • 42 Berlin
  • Codam / 42 Amsterdam Discovery And Piscine
  • Practical Checks: broader 42 campus comparison

4. Optional Finance Check

Ireland IFS

This is worth checking because it is closer to the Investment20/20 idea, but may be easier than a UK route if work-right is an issue.

First check:

  • Confirm whether the application window is open.
  • Check eligibility and start dates.
  • Decide whether this kind of finance apprenticeship actually sounds interesting enough.

Useful links:

  • Ireland International Financial Services Apprenticeship
  • LIA / Insurance Practitioner Apprenticeship Ireland
  • Investment20/20 Trainee Programme
