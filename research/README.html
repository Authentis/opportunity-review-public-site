Research System

Audience: Shareable only with care.

This directory is for ongoing research about how to help a young adult.

The goal is not to collect as many links as possible. The goal is to collect a small number of relevant opportunities and organize them so a human can review them quickly.

Principles

  • Prefer fewer, better opportunities.
  • Avoid duplicates.
  • Keep raw findings separate from digest summaries.
  • Write for human decision-making, not for archival completeness.
  • Always explain why an item may fit a young adult specifically.

Current Research Priorities

  • Fintech is a priority theme.
  • Sailing is a priority theme.
  • These are priorities, not hard constraints.
  • The search should stay broad enough to include other strong opportunities that offer:
    structure, stronger peers, confidence-building, or practical momentum.

Structure

  • master_tracker.md
    Single table of active opportunities and research leads.
  • digests
    Human-readable summaries of the best findings from a run or period.
  • briefs
    Short research briefs used to align later search phases with the current working model.
  • opportunities
    One file per serious opportunity.
  • templates
    Templates for new entries.

Workflow

  1. New research run finds a small set of relevant items.
  2. Each serious item gets its own note in opportunities/.
  3. master_tracker.md is updated with status, fit, deadlines, and links to detailed notes.
  4. A digest file summarizes the best findings in plain language.

Quality Bar

An item is worth keeping only if it is at least one of:

  • a concrete opportunity with a real application path,
  • a strong environment worth joining,
  • a program that adds structure or peers,
  • or a practical option that reduces drift and increases confidence.
